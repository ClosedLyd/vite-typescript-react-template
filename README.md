## Vite + TypeScript + React Template

This is a starter template for building React applications using [Vite](https://vitejs.dev/), [TypeScript](https://www.typescriptlang.org/), and [React](https://react.dev/).

Has configured linting setup using [oxlint](https://oxc.rs/docs/guide/usage/linter.html)

### Features

- Fast development with Vite
- Type safety with TypeScript
- Modern React setup
- Fast linting and formatting

### Getting Started

1. **Install dependencies:**

   ```sh
   npm install
   ```

2. **Start the development server:**

   ```sh
   npm run dev
   ```

3. **Build for production:**

   ```sh
   npm run build
   ```

4. **Preview the production build:**
   ```sh
   npm run preview
   ```

### Project Structure

```
src/
  main.tsx        # Entry point
  App/            # Main App component
public/           # Static assets
vite.config.ts    # Vite configuration
tsconfig.json     # TypeScript configuration
.oxfmtrc.json     # Oxlint configuration
```
